# T04 — billing 主表格与批量催款区实现

## 完成内容

### 1. BillingTable.vue — 8 列案件收费表格
- 展示列：checkbox / 案件名称 / 客户 / 所属 Group / 应收(¥) / 已收(¥) / 未收(¥) / 下一收款节点 / 回款状态
- 逾期行背景高亮（rgba 红色底色）
- 仅 overdue 行 checkbox 可勾选，其余状态 disabled + 降低透明度
- 空状态展示"暂无收费记录"
- 响应式：md 隐藏客户列与节点列，lg 隐藏 Group 列

### 2. BillingBulkActionBar.vue — 批量操作栏
- 仅在 selectedCount > 0 时显示
- 显示已选计数、清除按钮、"批量生成催款任务"按钮
- 使用 shared/ui/Button 组件

### 3. useBillingSelection.ts — 行选择状态管理
- `isRowSelectable(row)` — 仅 overdue 返回 true
- `selectableRows(rows)` — 过滤可选行
- `toggleAll/toggleRow/clearSelection` — 全选/单选/清除
- `isAllSelected/isIndeterminate` — 计算全选与半选状态

### 4. useBillingBulkActions.ts — 批量催款逻辑
- `canCollect(row)` — 判断行是否可催款
- `executeBulkCollection(selectedIds, rows)` — 返回三段式 CollectionResult
- 逾期行标记 success + taskId，非逾期行标记 skipped + not-overdue

### 5. 测试覆盖
- useBillingSelection.test.ts — 14 个测试用例
- useBillingBulkActions.test.ts — 11 个测试用例
- 覆盖：选择机制、禁用逻辑、全选/半选、批量结果生成
