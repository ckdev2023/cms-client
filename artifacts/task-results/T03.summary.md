# T03 — billing 摘要卡与筛选区实现

## 完成内容

### 1. `BillingSummaryCards.vue` — 摘要卡组件
- 接收 `cardDefs`（卡片定义）和 `summary`（摘要数据）两个 props，均可选带默认值。
- 4 张卡片分别展示：总应收 / 总已收 / 总未收 / 逾期欠款。
- 支持 3 种 variant（default / primary / danger），逾期卡片使用 danger 色调。
- 金额使用 `toLocaleString("ja-JP")` 格式化为 ¥ 显示。
- 响应式网格：1 列 → 2 列 → 4 列。

### 2. `BillingFilters.vue` — 筛选区组件
- 使用 `SegmentedControl` 切换两段视图（案件收费列表 / 回款流水记录），无"发票管理"。
- 搜索框使用 `SearchField`（inline 变体），支持案件名称/客户名称/案件编号。
- 3 个下拉筛选器：回款状态 / Group / 负责人。
- 重置按钮仅在筛选激活时显示。
- 底部可选显示筛选结果计数。
- 所有新增 props 带默认值，兼容 T02 总装壳。

### 3. `useBillingFilters.ts` — 筛选 model
- 在 T02 骨架基础上新增 `segment` ref 和 `switchSegment` 方法。
- 保留原有 `statusFilter` / `groupFilter` / `ownerFilter` / `search` / `isFilterActive` / `resetFilters` / `applyFilters`。
- `resetFilters` 清除所有筛选但保留 segment 选择。

### 4. `useBillingFilters.test.ts` — 19 项测试
- 默认值初始化（segment / filters / isFilterActive）
- isFilterActive 对每个筛选字段的响应
- resetFilters 清除筛选但保留 segment
- switchSegment 切换分段
- applyFilters 按 status / group / owner / search 过滤
- 搜索匹配 caseName / clientName / caseNo，且大小写不敏感
- 多条件交集过滤
- 无匹配返回空
- deps 透传
