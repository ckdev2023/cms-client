# T06 — billing 页面总装与路由替换

## 变更摘要

将 `/billing` 从占位页切换为完整可渲染的收费列表页面。

## 修改文件

| 文件 | 变更类型 | 说明 |
|------|----------|------|
| `packages/admin/src/views/billing/BillingListView.vue` | 重写 | 从骨架壳升级为完整总装层 |
| `packages/admin/src/router/index.ts` | 修改 | `/billing` 路由指向 BillingListView |

## BillingListView 总装内容

- **composable 串联**：useBillingFilters、useBillingSelection、useBillingBulkActions、useBillingToast
- **fixtures 数据注入**：SAMPLE_BILLING_ROWS、SAMPLE_SUMMARY、SUMMARY_CARD_DEFS、BILLING_SEGMENTS、SAMPLE_PAYMENT_LOGS、SAMPLE_BILLING_PLANS
- **摘要卡**：传入 cardDefs + summary
- **筛选区**：双向绑定 segment / search / statusFilter / groupFilter / ownerFilter，传入 segments + options + filteredCount + isFilterActive
- **分段视图**：billing-list 显示表格 + 批量栏，payment-log 显示回款流水
- **主表格**：传入 filteredRows + selectedIds + isAllSelected + isIndeterminate，监听 toggle-row / toggle-all
- **批量催款栏**：传入 selectedCount + loading，监听 clear / bulk-collect
- **回款流水表**：传入 entries，监听 void-payment / reverse-payment
- **分页**：根据当前 segment 显示对应总数
- **登记回款弹窗**：传入 open + nodes，监听 close / submit
- **toast**：传入 visible + title + description，监听 dismiss

## 路由变更

- 从 `placeholderRoutes` 数组中移除 `/billing` 条目
- 新增显式路由：`{ path: '/billing', name: 'billing', component: () => import('BillingListView.vue'), meta: { navKey: 'billing', groupKey: 'finance' } }`
