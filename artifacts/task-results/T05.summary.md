# T05 — billing 回款流水与登记回款弹窗

## 完成内容

### 组件

- **PaymentLogTable.vue** — 8 列回款流水表（金额/日期/案件/节点/凭证/状态/操作人/备注）+ 操作列。
  已作废/已冲正行以 muted 色调 + 金额删除线展示；有效行提供「作废」「冲正」操作入口。
- **PaymentModal.vue** — 登记回款弹窗，5 个字段（金额/日期/关联节点/凭证/备注）。
  内部使用 `usePaymentModal` composable；金额超限为软提示（不阻断），多未结清节点未选择时阻断提交。
  使用 `Teleport` 渲染到 body，overlay 点击可关闭。
- **BillingToast.vue** — 固定右下角 toast 通知，props 驱动（visible/title/description），
  带入场/离场过渡动画。

### Model

- **usePaymentModal.ts** — 增强版：提取 `emptyForm()` 和 `resolveNodes()` 辅助函数；
  新增 `parsedAmount`、`selectedNode`、`amountExceedsNode`（软警告）、`needsNodeSelection`（阻断提交）；
  `open()` 自动过滤已结清节点，仅剩一个时自动选中。
- **useBillingToast.ts** — 已由 T02 实现，无需修改。
- **useRiskAckLog.ts** — 已由 T02 实现，无需修改。

### 测试

- `usePaymentModal.test.ts` — 22 个用例覆盖：初始状态、open 过滤 paid 节点、自动选中单节点、
  canSubmit 校验（金额/日期/节点）、needsNodeSelection 阻断、amountExceedsNode 软警告、
  selectedNode 关联、parsedAmount 解析、close 重置、resetForm。
