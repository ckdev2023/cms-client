# T08 — dashboard 到 billing 的入口联动

## 变更概要

将 dashboard 中两个 billing 相关入口统一接到真实 `/billing` 页面：

1. **WorkPanelSection — billingReceipt 工作项**：在 `workPanelData.ts` 的 `WorkItemDef` 接口新增可选 `route` 字段，`billingReceipt` 条目设置 `route: "/billing"`。`WorkPanelSection.vue` 引入 `useRouter`，点击动作按钮时如果工作项携带 `route` 则执行 `router.push`。

2. **QuickActionsPanel — addReceipt 内联动作**：将 `inlineActions` 从纯字符串数组改为 `{ id, route? }` 对象数组，`addReceipt` 入口设置 `route: "/billing"`，点击时导航到收费列表。

## 修改文件

| 文件 | 说明 |
|------|------|
| `workPanelData.ts` | `WorkItemDef` 新增 `route?: string`；`billingReceipt` 设置路由 |
| `WorkPanelSection.vue` | 引入 `useRouter`，按钮 click 导航 |
| `QuickActionsPanel.vue` | 引入 `useRouter`，inlineActions 改为对象数组 |
| `WorkPanelSection.test.ts` | 补充 vue-router mock |
| `QuickActionsPanel.test.ts` | 补充 vue-router mock |
