# T07 — billing i18n 文案接入

## 完成内容

为 billing 模块接入 zh-CN / en-US / ja-JP 三语文案，消除页面中所有硬编码 UI 文案。

## 主要变更

### 新建文件
- `packages/admin/src/i18n/messages/billing/zh-CN.ts`
- `packages/admin/src/i18n/messages/billing/en-US.ts`
- `packages/admin/src/i18n/messages/billing/ja-JP.ts`

### Locale root 合并
- 在 `zh-CN.ts` / `en-US.ts` / `ja-JP.ts` 三个 root 文件中导入并挂载 `billing` 命名空间，不改动其他内容。

### 组件 i18n 替换（9 个组件）
| 组件 | 替换项 |
|------|--------|
| BillingListView | toast 标题与描述 |
| BillingSummaryCards | aria-label、卡片 label（通过 labelKey） |
| BillingFilters | 分段标签、搜索占位、筛选 aria-label、下拉默认项、重置按钮、记录计数 |
| BillingTable | 表头、全选 aria-label、行选择 aria-label、状态标签、Group 标签、空态、节点空态 |
| BillingBulkActionBar | aria-label、选中计数文案、清除按钮、催款按钮 |
| PaymentModal | 标题、aria-label、提示、所有字段标签/占位/错误/警告、取消/提交按钮 |
| PaymentLogTable | 表头、凭证有/无、记录状态、操作按钮、空态 |
| BillingToast | 关闭 aria-label |
| BillingPagination | 分页摘要、上一页/下一页 |

### fixtures / types 适配
- `BillingSummaryCardDef.label` → `labelKey`（i18n 键）
- `BILLING_SEGMENTS[].label` → `labelKey`
- `BILLING_STATUS_OPTIONS[].label` → i18n 键
