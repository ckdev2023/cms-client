# 在留资格别必要资料清单 Ver2 - 中文规范版

## 文档说明

- 来源：`docs/事务所流程/在留資格別必要情報一覧Ver2.xlsx`
- 编译依据：`docs/事务所流程/在留資格別必要情報一覧Ver2.ai-optimized.md`
- 目标：把原始日文矩阵整理成可直接执行的中文资料清单
- 状态定义：
  - `必需`：默认必须收集
  - `条件必需`：满足条件时必须收集
  - `未列出`：源表未列出，不代表绝对不需要

## 全局执行规则

1. 先确定主场景，只能从 7 个场景中选 1 个主场景。
2. 先收全局共通硬必需项：`护照复印件`。
3. 再收当前场景的全部必需项。
4. 再判断条件必需项是否触发。
5. 最后检查是否存在 `日本`、`本国・日本`、`需翻译` 等额外备注。

## 1. 经营管理签认定 4 个月

### 必需材料

- 护照复印件（`パスポート写し`）
- 证件照 3x4 厘米（`証明写真`）
- 经历书 / 履历书（`経歴書・履歴書`）
- 毕业证明（本国/日本，外国文件需翻译）
- 营业许可证
- 资格证明（日语能力、执照等）
- 在职 / 离职证明等工作经历证明
- 资本金 3000 万日元以上余额证明
- 资本金来源证明 / 收入证明
- 现住址信息
- 来日履历
- 最近出入境日期
- 在留资格认定申请记录
- 在留资格认定不许可记录
- 日本亲属情况
- 公司名称及事业目的预定
- 事务所预定物件资料
- 事务所设置委任状（含受任人电话）

### 条件必需材料

- 结婚证明（仅家族滞在时）
- 出生证明（仅家族滞在时）
- 其他视个案需要的材料

### 执行重点

- 重点检查资本证明链是否完整。
- 重点确认来日履历与既往认定 / 不许可历史。
- 事务所相关资料虽然不是正式租约，但不能缺前置准备证据。

- 其他视个案需要的材料

- 该场景是“前置设立包”，不要误当成签证申请完整包。

## 7. 企业内转勤

### 必需材料

- 护照复印件
- 证件照 3x4 厘米
- 经历书 / 履历书
- 在职 / 离职证明等工作经历证明
- 收入相关资料
- 公司登本（履历事项全部证明书，3 个月内）
- 最近决算报告书或个人确定申告书
- 前一年工资所得源泉征收票等法定调书合计表
- 转勤命令
- 转让 / 让渡合同复印件
- 股东名簿

### 条件必需材料

- 其他视个案需要的材料

### 特别备注

- `最近决算报告书或个人确定申告书` 标注了 `本国・日本`。
- `股东名簿` 标注了 `日本`。

### 执行重点

- 必须证明集团内部调动关系，而不是普通招聘。
- 日本侧和本国侧材料要能互相对应，不能只交单边证据。

## 收口检查

- 是否已确定唯一主场景
- 是否已覆盖该场景全部必需材料
- 是否已逐条判断全部条件必需材料
- 是否已处理全部备注要求
- 是否已标记缺失项、责任人和补件截止时间
