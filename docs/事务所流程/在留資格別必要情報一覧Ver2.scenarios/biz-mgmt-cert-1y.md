# 经营管理签认定 1 年资料清单

- `scenario_id`: `biz_mgmt_cert_1y`
- 日文来源名：`経営管理 / 認定1年`
- 来源：`在留資格別必要情報一覧Ver2.xlsx`

## 适用说明

适用于公司实体已基本落地后的经营管理签认定 1 年场景，是经营管理相关场景中材料最完整的一类。

## 必需材料

- 护照复印件
- 证件照（3x4 厘米）
- 经历书 / 履历书
- 毕业证明（本国/日本，外国文件需翻译）
- 营业许可证
- 资格证明（日语能力、执照等）
- 在职 / 离职证明等工作经历证明
- 资本金 3000 万日元以上余额证明
- 资本金来源证明 / 收入证明
- 现住址信息
- 来日履历
- 最近出入境日期
- 在留资格认定申请记录
- 在留资格认定不许可记录
- 日本亲属情况
- 住民票
- 印鉴证明（公证书）
- 手机号码
- 在留卡
- 事务所租赁合同
- 事务所照片
- 事务所平面图（面积）
- 公司登本（履历事项全部证明书，3 个月内）
- 各类设立申报（国税 / 府税 / 市税）
- 定款
- 役员报酬股东会决议记录
- 交易方名片或合同

## 条件材料

- 结婚证明（仅家族滞在时）
- 出生证明（仅家族滞在时）
- 其他视个案需要的材料

## 重点注意

- 需同时覆盖申请人背景、办公场地、公司主体、税务设立四条证据链。
- 事务所实体材料和公司法定材料容易成为漏项。
