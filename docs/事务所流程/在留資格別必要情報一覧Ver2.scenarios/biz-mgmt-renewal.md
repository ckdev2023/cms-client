# 经营管理签续签资料清单

- `scenario_id`: `biz_mgmt_renewal`
- 日文来源名：`経営管理 / 期間更新`
- 来源：`在留資格別必要情報一覧Ver2.xlsx`

## 适用说明

适用于经营管理签期间更新场景，重点不在设立，而在持续经营与持续在留事实。

## 必需材料

- 护照复印件
- 证件照（3x4 厘米）
- 现住址信息
- 日本亲属情况
- 住民票
- 最近课税证明 / 纳税证明
- 手机号码
- 在留卡
- 公司登本（履历事项全部证明书，3 个月内）
- 前一年工资所得源泉征收票等法定调书合计表
- 雇用保险适用事业者编号
- 交易方名片或合同

## 条件材料

- 其他视个案需要的材料

## 重点注意

- 续签材料要能证明公司在持续经营。
- 如果只有存续证明，没有税务和业务往来证明，材料链会不完整。
