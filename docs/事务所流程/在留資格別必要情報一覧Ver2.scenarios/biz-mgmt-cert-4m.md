# 经营管理签认定 4 个月资料清单

- `scenario_id`: `biz_mgmt_cert_4m`
- 日文来源名：`経営管理 / 認定4か月`
- 来源：`在留資格別必要情報一覧Ver2.xlsx`

## 适用说明

适用于经营管理签认定 4 个月场景，重点在申请人背景、资本来源、来日履历与创业前置准备。

## 必需材料

- 护照复印件
- 证件照（3x4 厘米）
- 经历书 / 履历书
- 毕业证明（本国/日本，外国文件需翻译）
- 营业许可证
- 资格证明（日语能力、执照等）
- 在职 / 离职证明等工作经历证明
- 资本金 3000 万日元以上余额证明
- 资本金来源证明 / 收入证明
- 现住址信息
- 来日履历
- 最近出入境日期
- 在留资格认定申请记录
- 在留资格认定不许可记录
- 日本亲属情况
- 公司名称及事业目的预定
- 事务所预定物件资料
- 事务所设置委任状（含受任人电话）

## 条件材料

- 结婚证明（仅家族滞在时）
- 出生证明（仅家族滞在时）
- 其他视个案需要的材料

## 重点注意

- 资本证明链必须可追溯。
- 来日履历、既往认定和不许可历史不能漏。
- 事务所相关材料属于前置准备证明，不应空缺。
