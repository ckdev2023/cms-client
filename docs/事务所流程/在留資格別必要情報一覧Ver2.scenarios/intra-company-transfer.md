# 企业内转勤资料清单

- `scenario_id`: `intra_company_transfer`
- 日文来源名：`企業内 / 転勤`
- 来源：`在留資格別必要情報一覧Ver2.xlsx`

## 适用说明

适用于集团或关联企业内部的人事调动场景，重点证明转勤关系，而不是普通招聘。

## 必需材料

- 护照复印件
- 证件照（3x4 厘米）
- 经历书 / 履历书
- 在职 / 离职证明等工作经历证明
- 收入相关资料
- 公司登本（履历事项全部证明书，3 个月内）
- 最近决算报告书或个人确定申告书
- 前一年工资所得源泉征收票等法定调书合计表
- 转勤命令
- 转让 / 让渡合同复印件
- 股东名簿

## 条件材料

- 其他视个案需要的材料

## 特别备注

- `最近决算报告书或个人确定申告书` 需关注 `本国・日本` 备注。
- `股东名簿` 需关注 `日本` 备注。

## 重点注意

- 日本侧与本国侧材料必须能互相对应。
- 如果缺少集团关系或调令证据，容易被理解成普通雇佣。
