# 技人国认定资料清单

- `scenario_id`: `eng_humanities_intl_cert`
- 日文来源名：`技人国 / 認定`
- 来源：`在留資格別必要情報一覧Ver2.xlsx`

## 适用说明

适用于技术・人文知识・国际业务签证认定场景，重点在学历、职历、雇佣条件和雇主主体证明。

## 必需材料

- 护照复印件
- 证件照（3x4 厘米）
- 经历书 / 履历书
- 毕业证明（本国/日本，外国文件需翻译）
- 学位证明（本国/日本，外国文件需翻译）
- 成绩证明（本国/日本，外国文件需翻译）
- 资格证明（日语能力、执照等）
- 在职 / 离职证明等工作经历证明
- 现住址信息
- 来日履历
- 日本亲属情况
- 最近课税证明 / 纳税证明
- 手机号码
- 在留卡
- 公司登本（履历事项全部证明书，3 个月内）
- 最近决算报告书或个人确定申告书
- 前一年工资所得源泉征收票等法定调书合计表
- 雇用保险适用事业者编号
- 员工列表
- 雇用条件通知书
- 公司介绍（如有）

## 条件材料

- 结婚证明（仅家族滞在时）
- 出生证明（仅家族滞在时）
- 在留申请在线回执
- 其他视个案需要的材料

## 重点注意

- 学历链、职历链和雇佣条件链必须同时完整。
- 雇主公司主体材料不足时，整体说服力会明显下降。
