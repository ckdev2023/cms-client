## 页面文档写作与读取约束

- 页面文档仍是**页面字段、Tab、动作与边界**的唯一权威正文
- 每篇页面正文在概览表后必须补 `## 0. 结构化速查（人 / AI 共用）`，至少包含：页面主对象、页面主动作、直接依赖、上游权威、本页不负责
- 结构化定位时，先读 [09-结构化总索引与交叉映射](../09-结构化总索引与交叉映射.md) 确认：
  - 该页面承载哪些主对象
  - 上游规则来自 03 / 04 / 07 的哪一段
  - 是否存在 P0 明确排除项，禁止页面侧脑补
- 页面规格不得重复发明业务规则；规则必须回链到 03 / 04 / 07
