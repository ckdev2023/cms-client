## 1. 典型任务

- **查看今日待办**：确认今天必须处理的任务、跟进、催办
- **识别风险案件**：快速发现有硬阻断、逾期资料、补正临近到期或欠款风险的案件
- **推进待提交案件**：定位已通过校验的案件，完成复核（如启用）和提交
- **跟进待回款**：确认收费节点已到但未结清的案件
- **切换视角**：在"我的/本组/全所"之间切换聚合范围

## 2. 核心卡片

### 2.1 P0 卡片清单

| 卡片 | 定义 | 数据来源 |
|------|------|---------|
| 今日待办 | 今日必须处理的动作集合 | Task.due_at=今日且 status≠done |
| 即将到期案件 | 关键期限将在 7/30 天内到期 | CaseDeadline.due_at 在窗口内 |
| 待补件案件 | 有资料项缺件或退回补正 | DocumentRequirement.status=missing/revision_required |
| 待提交案件 | 满足提交 Gate 的案件 | Case.stage=S6 + latest_validation_status=passed |
| 审理中案件 | 已提交等待结果 | Case.stage=S7 |
| 待回款案件 | 收费节点已到但未结清 | BillingPlan.status=due 且 amount_due>amount_paid |
| 风险案件 | 存在硬阻断或关键风险 | has_blocking_issue_flag 或 blocking_count>0；资料逾期；欠款风险 |

> P0 不做"员工负载"卡片和"本月新增案件"卡片——这两项后置到 P1。

### 2.2 默认筛选

- 聚合范围：我的 / 本组 / 全所（管理员）
- 时间窗口：7 天 / 30 天（即将到期卡片可切换）

## 3. 卡片详情（核心区块）

### 今日待办

- **行动入口**：批量指派/批量完成、一键进入案件资料清单/补正任务

### 即将到期案件

- **行动入口**：一键创建跟进任务

### 待提交案件

- **行动入口**：进入"校验与提交"Tab、生成提交包、上传回执/凭证

### 风险案件

- **行动入口**：一键定位阻断项、批量指派责任人

### 待回款案件

- **行动入口**：登记回款、上传凭证、创建催款任务

## 4. 关键动作

| 动作 | 触发条件 | 结果 | 权限 |
|------|---------|------|------|
| 切换聚合范围 | 点击"我的/本组/全所"切换 | 所有卡片计数与列表同步刷新 | 全所仅管理员 |
| 批量完成待办 | 勾选多条今日待办 | 批量标记完成 | 任务负责人/管理员 |
| 钻入案件详情 | 点击卡片中的案件 | 跳转案件详情页 | 有案件可见权限 |
| 一键创建跟进任务 | 从即将到期卡片发起 | 创建任务并关联案件 | 主办人、助理 |
| 登记回款 | 从待回款卡片发起 | 跳转收费 Tab 或弹窗录入 | 财务、主办人 |

## 5. 状态与异常

- **空状态（无待办）**：显示"暂无待办，去查看全部案件"引导
- **无权限查看全所**：隐藏"全所"切换选项，默认显示"我的"
- **数据加载中**：卡片骨架屏，避免大量聚合查询阻塞交互
- **卡片计数为零**：卡片仍显示但计数为 0，不隐藏

## 6. 权限与可见性

| 角色 | 可见范围 | 可操作范围 |
|------|---------|-----------|
| 管理员 | 全所聚合 | 全部卡片行动入口 |
| 主办人 | 本组 + 负责/协作的案件 | 对应卡片行动入口 |
| 助理 | 本组 + 协作案件 | 待办、催办相关 |
| 财务 | 全所（仅收费相关卡片） | 待回款卡片 |

切换范围时需同步影响列表与计数。

## P0 明确不做

| 不做项 | 后置到 | 原因 |
|--------|-------|------|
| 看板视图 | P1 | P0 用列表+基础聚合 |
| 日历视图 | P1 | P0 用列表+基础聚合 |
| 自定义仪表盘 | P1 | 首版用固定卡片布局 |
| 高级报表 | P2 | 首版不需要复杂统计与分析 |
| 员工负载卡片 | P1 | 首版不需要复杂人力分析 |

## 附：角色化默认卡片（P0 简化版）

P0 所有角色共享统一卡片布局，不做角色化视图切换。各角色的差异体现在数据可见性上：

| 角色 | 主要关注的卡片 |
|------|--------------|
| 行政书士/主办人 | 待提交、风险案件、审理中、待回款 |
| 助理 | 今日待办、待补件、即将到期 |
| 财务 | 待回款 |
