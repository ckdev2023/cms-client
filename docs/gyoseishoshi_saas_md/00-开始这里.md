## 关键概念速览

在深入文档前，先了解以下核心对象（详细定义见 [08-术语表](08-术语表.md)）：

- **Customer**（个人客户）— 客户主数据，一个客户可关联多个案件
- **Case**（案件）— 系统核心工作单元，一案一单
- **Group**（归属团队）— 内部协作归属维度，非企业客户
- **Stage**（阶段 S1–S9）— 案件所处的流程阶段
- **Gate**（门槛校验）— 阶段间的质量关卡（A/B/C 三道）
- **DocumentRequirement**（资料项）— "需要什么材料"的要求
- **SubmissionPackage**（提交包）— 一次对外提交的完整快照

```
Lead ──转化──→ Customer ──建案──→ Case
                                    │
                    ┌───────────────┼───────────────┐
                    │               │               │
              CaseParty    DocumentRequirement   BillingPlan
              (关联人)      (资料项)              (收费节点)
                                │                     │
                        DocumentFileVersion    PaymentRecord
                        (附件版本)              (回款记录)
                                │
                        SubmissionPackage ←── GeneratedDocumentVersion
                        (提交包)              (文书版本)
```
