# 03 MVP 功能优先级

## 1. MVP 目标

MVP 的目标不是一次性覆盖全部事务所场景，而是优先打通案件主链路：

**建档 → 收资料 → 催资料 → 做文书 → 校验 → 跟进 → 回款 → 留痕**

---

## 2. P0（首版必须上线）

### 2.1 客户管理
- 个人客户 / 企业客户创建
- 客户基础字段
- 关联案件
- 基础搜索与筛选

### 2.2 案件管理
- 创建案件
- 绑定客户 / 企业 / 关联人
- 案件编号
- 案件状态
- 负责人/协作者
- 截止日期

### 2.3 资料清单管理
- 案件模板生成资料清单
- 资料状态切换
- 附件上传
- 审核意见
- 完成率展示

### 2.4 缺件催办
- 待催办列表
- 手动发送催办
- 自动提醒规则基础版
- 催办记录

### 2.5 沟通记录
- 手工记录电话/消息/面谈/邮件摘要
- 案件时间线
- 内部备注

### 2.6 模板文书生成
- 模板库基础版
- 变量填充
- 导出 Word / PDF

### 2.7 期限提醒
- 在留到期提醒
- 补件截止提醒
- 提交截止提醒
- 负责人提醒

### 2.8 收费与回款
- 费用总额
- 收费节点
- 回款记录
- 欠款标记

### 2.9 审计日志
- 关键操作记录
- 状态变更记录
- 上传与审核记录

---

## 3. P1（首版后优先增强）

- 客户在线上传资料
- 查看缺件
- 查看进度
- 查看付款状态

### 3.2 提交前校验
- 字段完整性检查
- 必需资料齐全检查
- 有效期检查
- 收费节点检查

### 3.3 多语言支持
- 中/日/英表单与提醒模板

### 3.4 高级工作台
- 看板视图
- 即将到期聚合
- 风险案件视图

### 3.5 文书版本管理
- 初稿 / 审核版 / 定稿
- 模板版本历史

### 3.6 续签商机自动创建
- 90/60/30 天提醒
- 自动创建续签跟进任务

---

### 4.2 自动化流程引擎

- 自动分派任务

### 4.3 高级权限系统
- 字段级权限
- 审批流
- 模板审批

### 4.4 外部集成

- SMS / LINE / WhatsApp

---

## 5. MVP 成功标准

如果首版上线后能做到以下几点，就说明 MVP 是成功的：

- 事务所可以不用 Excel 主表管理案件
- 案件资料清单可在线追踪
- 缺件可被系统提醒和留痕
- 模板文书能自动生成基础文档
- 到期案件不再依赖人工记忆
- 收费和案件进度能对应起来
- 至少一个事务所能完整跑通从建档到归档的流程
