## 2. i18n / Console 复验

### R-H5-I1（硬编码 aria-label）

R-H5-1 发现 `AppShell.vue` 中 `aria-label="关闭导航遮罩"` 硬编码中文在 ja-JP locale 下泄露。

R-H5-2 复验：所有 CJK aria-label 均为日文（ja locale），无中文泄露 ✅

```json
[
  {"tag": "BUTTON", "ariaLabel": "ナビゲーションを開く"},
  {"tag": "BUTTON", "ariaLabel": "グローバル検索"},
  {"tag": "SELECT", "ariaLabel": "表示言語"},
  {"tag": "ASIDE",  "ariaLabel": "サイドナビゲーション"},
  {"tag": "NAV",    "ariaLabel": "メインナビゲーション"},
  {"tag": "NAV",    "ariaLabel": "パンくずリスト"},
  {"tag": "DIV",    "ariaLabel": "案件詳細タブ"}
]
```

### R-H5-I2（`<html lang>` 不随 locale）

R-H5-1 发现 `<html lang="zh-CN">` 固定，浏览器 date input 文案与页面语言不一致。

```json
{"lang": "ja", "dataLocale": "ja-JP"}
```

- `lang="ja"`（短码，浏览器内置控件读取） ✅
- `data-locale="ja-JP"`（应用层完整码） ✅
