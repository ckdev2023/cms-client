# AI开发规范 - Vue 3版

## 目录结构

```text
vue3/
├── 00-核心规则卡片.md
├── 01-开发流程
├── 02-代码规范
├── 03-测试规范
├── 04-质量门禁
├── 05-协作规范
└── 06-提示词模板
```

## 推荐阅读顺序

1. `00-核心规则卡片.md`
2. `01-开发流程/AI开发完整流程.md`
3. `02-代码规范/Vue3标准代码参考.md`
4. `03-测试规范/测试分层策略.md`
5. `04-质量门禁/完整性检查清单.md`
6. `06-提示词模板/`

## 适用技术栈

- Vue 3
- TypeScript
- Vite
- Pinia
- Vue Router
- Vitest
- Vue Test Utils
- Playwright

## 使用建议

- 先读 `00` 快速对齐核心规则
- 再用 `01` 统一流程
- 用 `02` 约束代码写法
- 用 `03` 明确测试边界
- 用 `04` 做提交门禁
- 用 `05` 管理协作与设计对接
- 用 `06` 直接复制提示词给 AI
