### 6.1 资料登记命名口径

- P0 默认不上传文件到 SaaS 服务器
- 资料清单内统一使用：`登记资料（本地归档）`
- 行内说明统一使用：`已登记` / `待登记` / `本地归档相对路径`
- 如需保留上传类动作，仅能用于“上传回执”等提交后动作，不能用于资料清单主入口
- 页头 CTA、成功跳转按钮、进度文案都必须避免“上传资料”表述，统一改为“登记资料”

### 6.2 waived 口径

- `waived` 项从完成率分母剔除
- 但在资料清单中仍可见，灰色行 + "无需提供" badge + 原因 tooltip
- 操作留痕：原因码 + 自由文本 + 操作人 + 时间
