可见性配置

管理跨组建案与协作案件的可见性策略。配置变更不回溯影响既有协作/归属关系。

是否允许跨组建案

启用后，非管理员也可为非本组客户创建案件。跨组建案须留痕（操作人、时间、原因）。

是否允许负责人查看非本组协作案件

启用后，被指定为协作者的负责人可查看非本组案件信息。

此处仅控制配置开关。跨组建案/转组/线索转化改组时的原因采集与审计留痕属于业务流程域，不在此页面管理。

本地资料根目录

配置事务所级本地资料服务器根目录，供案件资料路径拼接使用。

根目录尚未配置

配置根目录后，案件/资料中心的「登记资料（本地归档）」功能才可正常使用。未配置期间该入口将被禁用。

仅管理员可见。请输入文件服务器的 UNC 路径或本地挂载点。

路径策略

系统仅保存 relative_path，禁止在业务对象中记录绝对路径。本地根目录在此处统一配置，业务模块自动拼接完整路径。

路径预览

\\fileserver\gyosei-docs/{relative_path}

实际完整路径 = 根目录 + 资料的相对路径
